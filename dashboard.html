Panduan Pembayaran Uang Kas

Sistem pembayaran kas kelas yang modern dan transparan kini hadir untuk memudahkan Anda.

Sistem pembayaran kas kelas dirancang untuk meningkatkan transparansi dan kemudahan dalam pengelolaan keuangan kelas. Anda dapat memantau kas pribadi, melihat tagihan, dan melakukan pembayaran langsung melalui website.

Login menggunakan NIM, akses menu Uang Kas, lalu bayar melalui QRIS. Setelah pembayaran, unggah bukti transaksi dan tunggu konfirmasi dari admin.

1. Login ke Website

  • Kunjungi
    • Tautan telah disalin!
  • Klik tombol Login
  • Masukkan NIM Anda untuk mengakses akun

Jika belum memiliki akun, hubungi admin/bendahara untuk pendaftaran.

2. Akses Menu x Uang Kas

  • Buka Dashboard kemudian pilih menu Uang Kas
  • Halaman ini menampilkan:
    • Total kas yang telah dibayar
    • Sisa tagihan yang belum dibayar
    • Progress kas kelas dalam persentase
    • Riwayat pembayaran lengkap

Dashboard Menu:

3. Pembayaran Melalui QRIS

  • Scroll ke bagian Poster Pembayaran Kas
  • Scan kode QRIS menggunakan aplikasi e-wallet atau mobile banking
  • Pastikan nominal pembayaran sesuai dengan jumlah kas yang akan dibayar
  • Simpan bukti transaksi setelah pembayaran berhasil

4. Unggah Bukti Pembayaran

  • Isi formulir Kirim Bukti Pembayaran dengan informasi berikut:
    • Nominal pembayaran
    • Tanggal pembayaran
    • Metode pembayaran (Transfer/Tunai)
  • Pilih file bukti transfer kemudian klik Upload
  • Tekan tombol Kirim Bukti Pembayaran
  • Status awal akan pending tunggu konfirmasi admin

5. Konfirmasi dari Admin

Setelah bukti pembayaran dikirim, admin akan melakukan verifikasi. Status pembayaran akan diperbarui secara otomatis setelah dikonfirmasi.

NOTE:

  • Pastikan nominal pembayaran sesuai dengan tagihan.
  • Bayar uang kas tepat waktu untuk mencapai target bulanan.
  • Hubungi admin jika mengalami kendala atau data tidak terupdate.
  • Jika mengalami kendala saat melakukan pembayaran, silakan hubungi admin melalui WhatsApp yang tertera di bawah ini.
